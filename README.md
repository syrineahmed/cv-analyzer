# CV Analyzer 🔍

An AI-powered tool that analyzes a CV against a job description and gives:
- A match score out of 10
- Top 3 strengths
- Top 3 weaknesses
- Specific advice to improve

## Built with
- Python
- Groq API (LLaMA 3.3)
- python-dotenv

## How to run
1. Clone the repo
2. Add your GROQ_API_KEY in a .env file
3. Run: python cv_analyzer.py
